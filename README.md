# Paradigma - Münazara Platformu

Bu proje, "Yapay Zeka Meslekleri Yok Eder" tezine karşı hazırlanan güçlü, felsefi ve ekonomik argümanları barındıran akademik bir münazara el kitabıdır.

## 🚀 Özellikler

- **Brutalist & Modern UI:** React ve Tailwind mantığıyla yazılmış sade tasarım.
- **Akademik Kaynakça:** Wikipedia tarzı atıf sistemi (`[1]`, `[2]`).
- **Karanlık Mod:** Göz yormayan, okuma odaklı gece modu.
- **Kopyala & Yapıştır:** Argümanları tek tıkla kopyalama desteği.
- **Print Mode:** Yazdırıldığında siyah-beyaz makale görünümüne geçiş.

## 🛠 Kurulum

Geliştirme ortamında çalıştırmak için:

```bash
npm install
npm run dev
```
